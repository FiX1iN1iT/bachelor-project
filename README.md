# Справочная вопрос-ответная система по медицине

Система позволяет задавать вопросы по содержимому медицинских PDF-документов и получать ответы на основе RAG-пайплайна. Ключевая особенность — весь ML-инференс (LLM + эмбеддинги) выполняется **целиком в браузере** через WebGPU и WebAssembly; содержимое документов и история чатов не покидают клиент.

---

## Архитектура

```
Браузер (весь ML-инференс)              Бэкенд (FastAPI)
──────────────────────────────          ─────────────────────────
React SPA + shadcn/ui           ←→      Аутентификация (JWT/bcrypt)
WebLLM (Qwen2.5-7B, WebGPU)            Метаданные документов (SQLite)
HuggingFace Transformers.js             Хранилище файлов (MinIO S3)
IndexedDB (векторное хранилище)
localStorage (чаты, документы)
```

---

## Требования

- **Браузер:** Chrome 113+ или Edge 113+ (поддержка WebGPU)
- **Docker:** для запуска MinIO
- **Node.js:** 18+
- **Python:** 3.10+
